# MoErgo Glove80 Custom Configuration for ZMK

Flashing instructions:

1. Turn off the keyboard and connect it via USB.
1. Power on keyboard while holding down Magic (1,1) and 'E' (-3,-3).
   > Mirrored for right side
1. Drag the file into the storage device (same file for both sides)
